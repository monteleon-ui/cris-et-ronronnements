# Contexte : cris-et-ronronnements

## 🗂️ Structure finale validée (PlanetHoster + Passenger)
```
nodevenv/
└── nocstorage/               # ✅ Dossier racine pour Node.js (imposé par PlanetHoster)
    ├── app.js                # Point d'entrée Express (PORT = process.env.PORT)
    ├── package.json          # Dépendances (main: "app.js")
    ├── package-lock.json
    ├── node_modules/         # Dépendances npm
    ├── noscript/             # pour les visiteurs qui ont interdit javascript
    │   ├── header.html
    │   ├── footer.html
    │   └── noscript.html
    ├── server/               # Non accessible au client
    │   ├── views/
    │   │   ├── layouts/main.ejs      # Layout principal
    │   │   ├── partials/             # Header, footer, noscript
    │   │   │   ├── header.ejs
    │   │   │   ├── footer.ejs
    │   │   │   └── noscript.ejs
    │   │   ├── pages/                # Pages EJS
    │   │   │   ├── index.ejs
    │   │   └── errors/               # Pages d'erreur
    │   │       ├── 404.ejs
    │   │       └── 500.ejs
    └── public/                # Accessible au client
        ├── assets/                  # CSS, JS, images, Bootstrap
        │   ├── Bootstrap/
        │   ├── css/
        │   ├── js/
        │   └── images/
public_html/                  # Dossier racine web (pour Passenger)
└── .htaccess                 # Configuration Passenger (voir ci-dessous)
```

## 📄 Fichiers clés

### `.htaccess` (dans `/public_html/`)
```apache
# DO NOT REMOVE. CLOUDLINUX PASSENGER CONFIGURATION BEGIN
PassengerAppRoot "/home/crisfahs/nodevenv/nocstorage"
PassengerBaseURI "/"
PassengerNodejs "/home/crisfahs/nodevenv/nocstorage/20/bin/node"
PassengerAppType node
PassengerStartupFile app.js
PassengerMinInstances 1
PassengerSpawnMethod direct
# DO NOT REMOVE. CLOUDLINUX PASSENGER CONFIGURATION END
```

### `app.js` (points critiques)
```javascript
// 1. Configurer EJS
app.set('view engine', 'ejs');
app.set('views', path.join(__dirname, 'server/views'));

// 2. Routes
app.get('/', (req, res) => { res.render('pages/index', { title: 'Accueil' }); });

// ... autres routes

// 4. Démarrage du serveur (PORT obligatoire pour Passenger)
const PORT = process.env.PORT || 3000;
app.listen(PORT, () => { console.log(`Server running on port ${PORT}`); });
```

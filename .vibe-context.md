# Contexte : cris-et-ronronnements

## 🎯 Objectif principal
Site web pour Simon Vitsonnet avec 3 parties :
- **BFK** : Questionnaire et dictionnaire (BDSM/Fetish/Kink)
- **IUFMM** : Cours pour Maîtres/Maîtresses
- **Éditions SV** : Œuvres littéraires (poèmes, livres)

## 🔧 Règles
1. **Jamais de données sensibles dans `/public/`**
2. Les pages d'erreur sont gérées par Express (via `server/errors/`)
3. Bootstrap et les assets publics restent dans `/public/assets/`
4. Le dossier `/dev/` est **exclu de Git** (pour développement local)

## 📦 Dépendances
- `express` : Serveur web
- `cookie-parser` : Gestion des cookies (contrôle parental)
- `gray-matter` + `marked` : Parsing Markdown

## 💻 Commandes
- Démarrer le serveur : `npm start`
- Vérifier l'état : `git status`

## 📚 Résumé du projet
- **Public cible** : Communauté LGBTQIA+ (comparable à Fetlife.com, mais pas exclusivement)
- **3 parties distinctes** :
  1. **Projet BFK** (BDSM, Fetish, Kink, etc.)
     - Questionnaire auto-administré (thèmes/groupes de questions)
     - Dictionnaire des questions (consultable indépendamment)
  2. **Projet IUFMM** (Institut Utile de Formation des Maîtres et Maîtresses)
     - Cours thématiques pour dominants/dominantes
     - Certains cours basés sur le dictionnaire BFK
  3. **Éditions SV** (Simon Vitsonnet Éditeur)
     - Œuvres en PDF/ePub/en ligne
     - Collections : "Rêveries" (poèmes), "BDSM" (ouvrages divers)

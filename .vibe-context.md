## 📄 Fichiers clés

### `.htaccess` (dans `/public_html/`)
```apache
# DO NOT REMOVE. CLOUDLINUX PASSENGER CONFIGURATION BEGIN
PassengerAppRoot "/home/crisfahs/nodevenv/nocstorage"
PassengerBaseURI "/"
PassengerNodejs "/home/crisfahs/nodevenv/nocstorage/20/bin/node"
PassengerAppType node
PassengerStartupFile app.js
PassengerMinInstances 1
PassengerSpawnMethod direct
# DO NOT REMOVE. CLOUDLINUX PASSENGER CONFIGURATION END
```

### `app.js` (points critiques)
```javascript
// 1. Configurer EJS
app.set('view engine', 'ejs');
app.set('views', path.join(__dirname, 'server/views'));

// 2. Middleware de contrôle parental (AVANT les routes)
app.use(parentalControlMiddleware);

// 3. Routes
app.get('/', (req, res) => { res.render('pages/index', { title: 'Accueil' }); });
app.get('/controle-parental', (req, res) => { res.render('pages/controle-parental', { title: 'Contrôle Parental' }); });
// ... autres routes

// 4. Démarrage du serveur (PORT obligatoire pour Passenger)
const PORT = process.env.PORT || 3000;
app.listen(PORT, () => { console.log(`Server running on port ${PORT}`); });
```

### `parental-control.js`
```javascript
module.exports = {
    enabled: true,
    minAge: 18,
    cookieName: 'ageVerified',
    cookieMaxAge: 30,
    protectedRoutes: ['/bfk', '/bfk/questionnaire', '/iufmm', '/sv_edition', '/reveries'],
    excludedRoutes: ['/controle-parental', '/', '/assets/', '/legal/', '/api/'],
    cookieOptions: {
        httpOnly: true,
        sameSite: 'lax',
        secure: true  // ✅ Obligatoire pour HTTPS en production
    }
};
```

## ⚠️ Problèmes courants et solutions

| Problème | Cause | Solution |
|----------|-------|----------|
| **"It works!" s'affiche** | Passenger charge une app par défaut | Corriger `.htaccess` + supprimer `public_html/index.html` |
| `npm command not found` | Node.js/npm non activé | `source /home/.../20/bin/activate` |
| Contrôle parental ne se déclenche pas | Routes protégées mal configurées | Vérifier `protectedRoutes` et `excludedRoutes` dans `parental-control.js` |
| Erreur ES9 (spread operator) | Environnement Passenger | Remplacer `...parentalConfig.cookieOptions` par les propriétés individuelles |
| `pm2 command not found` | PM2 non installé globalement | Utiliser le chemin absolu `/home/.../node_modules/.bin/pm2` |

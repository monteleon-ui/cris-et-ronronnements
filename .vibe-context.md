- **Sécurité** :
  Utilisation de `helmet` (en-têtes HTTP) et `morgan` (logging).

- **Structure des templates** :
  `main.ejs` comme layout principal, avec inclusion dynamique du contenu via `body`.

## 🌐 Configuration de production (PlanetHoster)
- **Hébergement** : PlanetHoster (World Premium) avec Passenger.
- **Node.js** : Version 20.x (LTS) dans `/nodevenv/nocstorage/20/bin/node`.
- **Passenger** : Utilisé pour exécuter Node.js (pas besoin de PM2 en production).
- **Structure** : `/nodevenv/nocstorage/` pour Node.js, `/public_html/.htaccess` pour Passenger.

## 🏗️ Décisions architecturales (mise à jour)

### Gestion des visiteurs sans JavaScript :
Balise `<noscript>` dans `main.ejs` pour rediriger vers [simonvitsonnet.com/cris-et-ronronnemets.html](https://simonvitsonnet.com/cris-et-ronronnemets.html).

### Sécurité :
- Utilisation de `helmet` pour les en-têtes HTTP de sécurité.
- **Nouveau** : `compression` middleware pour réduire la taille des réponses.
- **Nouveau** : CSP (Content Security Policy) renforcé avec restrictions strictes.
- **Nouveau** : HSTS (HTTP Strict Transport Security) activé.
- Utilisation de `morgan` pour le logging des requêtes.

### Structure des templates :
`main.ejs` comme layout principal, avec inclusion dynamique du contenu via `body`.

### Bootstrap :
- **Changement majeur** : Passage de Bootstrap **local (8.4M)** à **CDN (jsdelivr)**.
- Avantages : Réduction drastique de la taille du dépôt, meilleure performance (cache partagé).

### Système de build :
- **Nouveau** : Automatisation avec PostCSS (CSS) et Terser (JS).
- **Nouveau** : Minification automatique des assets via scripts npm.

## 🚀 Optimisations récentes (Juillet 2025)

### Performance ✅
- **Réduction de 94%** : Taille du dépôt passée de ~10.2M à ~536K.
- **Bootstrap en CDN** : Suppression des fichiers locaux (8.4M d'économie).
- **Minification** : CSS et JS automatiquement minifiés.
- **Compression HTTP** : Middleware `compression` ajouté.
- **Cache statique** : Configuration de cache long pour les assets (1 an).

### Sécurité ✅
- **0 vulnérabilités** : Audit npm propre.
- **CSP renforcé** : Content Security Policy avec restrictions strictes.
- **HSTS** : HTTP Strict Transport Security (1 an).
- **Dépendances à jour** : express@4.22.2, ejs@3.1.10, helmet@7.1.0.

### Accessibilité (RGAA) ✅
- **Skip to main content** : Lien de navigation pour les lecteurs d'écran.
- **Aria-label optimisés** : Suppression des redondances.
- **Structure sémantique** : Balises HTML5 correctement utilisées.
- **Contrastes vérifiés** : Toutes les couleurs ont un ratio ≥ 4.5:1.

### SEO ✅
- **Open Graph** : Balises pour un meilleur partage sur les réseaux sociaux.
- **Twitter Cards** : Optimisation pour Twitter.
- **Sitemap.xml** : Création d'un sitemap complet.
- **Robots.txt** : Configuration pour les crawlers.
- **Canonical URL** : Évite le contenu dupliqué.

### Build ✅
- **PostCSS** : Minification et autoprefixing du CSS.
- **Terser** : Minification du JavaScript.
- **Scripts npm** : `build`, `build:css`, `build:js`, `dev`, `start`.
- **Variables d'environnement** : Configuration via `.env` et `.env.example`.

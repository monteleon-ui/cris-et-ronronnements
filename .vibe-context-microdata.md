# Contexte : Microdonnées Schema.org pour cris-et-ronronnements

## 🎯 Objectif des microdonnées
Les **microdonnées Schema.org** permettent d'améliorer le **référencement (SEO)** et la **compréhension sémantique** du site par les moteurs de recherche (Google, Bing, etc.).

Pour **cris-et-ronronnements.net**, elles aident à :
- **Décrire clairement** les 3 parties du site (BFK, IUFMM, Éditions SV).
- **Améliorer l'affichage** dans les résultats de recherche (Rich Snippets).
- **Structurer le contenu** pour une meilleure indexation.

---

## 🌐 Informations globales du site
- **URL du site** : `https://cris-et-ronronnements.net`
- **Nom du site** : `Cris et Ronronnements`
- **Description** : *Site dédié à l'exploration de soi et des autres à travers des questionnaires, des ressources et des œuvres littéraires.*
- **Logo** : `/assets/images/favicon.ico` (ou une image dédiée si disponible).
- **Auteur** : Simon Vitsonnet
- **Pas de réseaux sociaux** : Aucun compte Facebook, Twitter, ou autre à lier.

## 📌 Règles pour les microdonnées

### 1. **Format à utiliser**
- **JSON-LD** (recommandé par Google) :
  ```html
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "TypeDeSchema",
    "propriété1": "valeur1",
    "propriété2": "valeur2"
  }
  </script>
  ```
- **Placement** : Toujours dans la balise `<head>` ou juste avant `</body>`.

## 🔍 Outils pour valider les microdonnées
1. **Google Rich Results Test** :
   - [https://search.google.com/test/rich-results](https://search.google.com/test/rich-results)
   - **Utilisation** : Copie-colle l'URL du site ou le code HTML.

2. **Schema Markup Validator** :
   - [https://validator.schema.org/](https://validator.schema.org/)
   - **Utilisation** : Copie-colle le code JSON-LD pour valider la syntaxe.

3. **Extension Chrome : Schema Markup Validator**
   - **Utilisation** : Installe l'extension et teste directement sur le site.

# Contexte : Microdonnées Schema.org pour cris-et-ronronnements

## 🎯 Objectif des microdonnées
Les **microdonnées Schema.org** permettent d'améliorer le **référencement (SEO)** et la **compréhension sémantique** du site par les moteurs de recherche (Google, Bing, etc.).

Pour **cris-et-ronronnements.net**, elles aident à :
- **Décrire clairement** les 3 parties du site (BFK, IUFMM, Éditions SV).
- **Améliorer l'affichage** dans les résultats de recherche (Rich Snippets).
- **Structurer le contenu** pour une meilleure indexation.

## 🔧 Règles pour les microdonnées

### 1. Format à utiliser
- **JSON-LD** (recommandé par Google) :
  ```html
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "TypeDeSchema",
    "propriété1": "valeur1",
    "propriété2": "valeur2"
  }
  </script>
  ```
- **Placement** : Toujours dans la balise `<head>` ou juste avant `</body>`.

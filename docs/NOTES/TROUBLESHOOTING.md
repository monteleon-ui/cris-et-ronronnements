## 🔍 **Comment Utiliser Ce Guide**

1. **Identifiez le symptôme** : Cherchez le message d'erreur ou le comportement inattendu dans la table ci-dessous.
2. **Consultez la solution** : Suivez les étapes proposées pour résoudre le problème.
3. **Testez** : Vérifiez que le problème est résolu.
4. **Documentez** : Si le problème n'est pas listé, ajoutez-le à ce fichier après l'avoir résolu.
